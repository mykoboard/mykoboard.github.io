# 🚀 Redefining Multiplayer: MyKoBoard is Here!

I'm excited to share a project that pushes the boundaries of web-based gaming and collaboration: **MyKoBoard**. 🎮

MyKoBoard isn't just another gaming platform; it's a **decentralized, serverless ecosystem** built for privacy, speed, and a premium user experience. No accounts, no central servers, just pure P2P interaction.

### 🛠️ The Tech Stack
We’ve leveraged a modern, high-performance stack to make this possible:
- **Frontend**: Vue 3 (Composition API) + Tailwind CSS for a sleek, glassmorphic UI.
- **P2P Networking**: WebRTC DataChannels for zero-latency, browser-to-browser sync.
- **State Management**: XState for predictable, complex game lifecycles.
- **Identity**: Cryptographic wallets for secure, user-owned sessions.
- **Infrastructure**: Strictly serverless using GitHub Pages and AWS Free Tier for signaling.

### 🌟 Why it Matters
In an era of data centralisation, MyKoBoard puts the power back in the users' hands:
- **Privacy-First**: Your data stays on your device (IndexedDB) and moves directly between peers.
- **Zero Infrastructure**: Built to scale without the overhead of traditional backends.
- **Versatile**: From classic games like Ludo and Tic-Tac-Toe to enterprise tools like Planning Poker and secure "War Rooms."
